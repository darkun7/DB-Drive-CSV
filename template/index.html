Database Drive CSV

How to Use

  1. Create file/ upload file to Google Drive
  2. There's 2 file extention that supported:
    • File with extension .CSV
      1. Send POST request as form data to url https://dbdrive.herokuapp.com/
      2. you may use example below:
        { "index": 1, "identifier": "ID" "csv": "https://drive.google.com/file/d/1xxk2ASmwnPeAym1or8tCl5Xi6QTcmAqh/view?usp=sharing" }
    • File with extension .XLS or .XLSX
      1. Send POST request as form data to url https://dbdrive.herokuapp.com/xls
      2. you may use example below:
        { "index": 1, "identifier": "ID" "xls": "https://docs.google.com/spreadsheets/d/1DM_-HOWXMsPmshnIq-wfJq7Jtc4pacLC6bUC8c1XzpU/edit?usp=sharing" }
